メッセージ


私は、高校を辞めた後、解体作業や足場を組む仕事をしながら半年間ほどその日暮らしをしておりました。
そんな折、親しい友人が逮捕され、その友人についてくださった国選付添人の熱心な弁護活動を見て、弁護士になることを決意し、晴れて弁護士となることができました。

私自身マイノリティに属しており、弁護士になれた以上、法的に困っている人・社会的に弱い立場の人を守るべく闘うことが私の使命だと考えております。
いい意味で弁護士らしくない弁護士として、紛争を解決すべく誠心誠意心を込めてお客様に寄り添った弁護活動をしていきたいです。

司法試験受験後、高校時代にやっていた総合格闘技を再開いたしました。
今後も格闘技を続けていきたいと考えており、これからは色々なものと闘う弁護士として精進していきたいと考えておりますので、よろしくお願いいたします。
